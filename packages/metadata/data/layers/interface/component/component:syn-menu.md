# syn-menu

## Summary

Menus provide a list of options for the user to choose from.

## Documentation

[Component Documentation](https://synergy-design-system.github.io/?path=/docs/components-syn-menu--docs)

## Class Information

- **Import Example:** `import SynMenu from '@synergy-design-system/components/components/menu/menu.js';`
- **Module Path:** components/menu/menu.js
- **Tag Name:** `syn-menu`

## Usage Information

- **Status:** stable
- **Since:** 2.0

## Available Slots

| Name      | Description                                                          |
| --------- | -------------------------------------------------------------------- |
| (default) | The menu's content, including menu items, menu labels, and dividers. |

## Available Events

| Name       | Event Type       | Description                           |
| ---------- | ---------------- | ------------------------------------- |
| syn-select | `SynSelectEvent` | Emitted when a menu item is selected. |
