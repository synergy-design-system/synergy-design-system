## Help Text

## Restricted

This restricts the combobox to only allow selections from the available options. Users cannot enter custom values that are not in the list.

```html
<syn-combobox
  value="Option 1"
  restricted=""
  size="medium"
  placement="bottom"
  form=""
  data-optional=""
  data-valid=""
>
  <syn-option
    value="option-1"
    role="option"
    aria-selected="false"
```
