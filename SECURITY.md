# Security Policy

## Supported Versions

We only support the latest released version of this project.

Security fixes are provided for the current version only. Users are expected
to update to the latest available release to receive security updates and
vulnerability fixes.

## Reporting a Vulnerability

If you believe you have identified a security vulnerability in this project,
please report it to the SICK Product Security Incident Response Team (PSIRT).

Please do not disclose vulnerabilities publicly until they have been assessed
and, where appropriate, remediated.

Please do not report security vulnerabilities through public GitHub issues,
discussions, or pull requests.
